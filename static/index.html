About the AI Model

This predictor uses a Linear Regression machine learning model trained on historical data of study hours and corresponding exam scores.

How it works: The model finds the best-fit line through data points (study hours vs. scores) and uses this line to make predictions for new inputs.

y = mx + b
Where: y = predicted score, x = study hours, m = slope, b = intercept

The algorithm has been trained on hundreds of data points to establish a reliable relationship between study time and performance.
